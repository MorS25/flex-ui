Alerts

Notice: the dismiss functionality requires the stripWicketTags markup option to be set to true. If this is not disabled manually, it will cause the outermost div container of the alert to remain in development mode.

Alert containers

Alert is a Border extensions that allows to display markup as an alert component. They allow for wrapping arbitrary html within an alert component. The alert css classes and the dismiss button is handled by the border component.
