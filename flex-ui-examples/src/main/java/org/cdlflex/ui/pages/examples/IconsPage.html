Icons

The IconType is a ICssClassNameProvider that can be extended for custom icon types 'in the bootstrap way'. It provides css classes for an Icon which is a WebMarkupContainer extension.

GlyphIcons

GlyphIconType is an extension of IconType that provides constants for the GlyphIcon set that comes with bootstrap.

FontAwesomeIcons

You will need to add the FontAwesomeCssResourceReference to your header to use FontAwesome icons.
